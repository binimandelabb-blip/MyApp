# рІерІѕріЋрїђрѕЇ рѕўріерѕІріерІФ рѕхрѕГрІЊрЅх рІерЅ░рїарЅЃрѕџ рѕўрѕўрѕфрІФ
# Community Crime Prevention System - User Guide

## рѕѕрѕЏрѕЁрЅарѕерѕ░рЅЦ ріарЅБрѕІрЅх / For Community Members

### рІерїЦрЅєрѕЏ рѕЏрѕхрїѕрЅбрІФ рѕўріЋрїѕрІХрЅй / Tip Submission Methods

#### 1. рІерѕърЅБрІГрѕЇ рѕўрЅ░рїЇрЅарѕфрІФ / Mobile Application

**ріЦріЋрІ┤рЅх рїЦрЅєрѕЏ ріЦріЋрІ░рѕџрІФрѕхрїѕрЅА:**

1. рѕўрЅ░рїЇрЅарѕфрІФрІЇріЋ рІГріГрЇѕрЅ▒ / Open the app
2. "рїЦрЅєрѕЏ рІФрѕхрїѕрЅА" / "Submit Tip" рІерѕџрѕѕрІЇріЋ рІГрїФріЉ
3. рІерІѕріЋрїђрѕЇ ріарІГріљрЅх рІГрѕЮрѕерїА / Select crime type:
   - рѕхрѕГрЅєрЅх (Theft)
   - рІўрѕерЇІ (Robbery)
   - рїЦрЅЃрЅх (Assault)
   - ріеріарІ░ріЋрІЏрІЦ ріЦрїй рїІрѕГ рІерЅ░рїѕріЊріў (Drugs)
   - ріарїарѕФрїБрѕф ріЦріЋрЅЁрѕхрЅЃрѕ┤ (Suspicious Activity)
   - рѕїрѕјрЅй (Others)

4. рІЮрѕГрІЮрѕГ рѕўрѕерїЃ рІФрѕхрїѕрЅА / Enter details:
   - рІерЅ░ріерѕ░рЅ░рЅарЅх рЅдрЅ│ (Location)
   - рІЮрѕГрІЮрѕГ рѕўрїЇрѕѕрїФ (Description)
   - рІеріарІ░рїІ рІ░рѕерїЃ (Urgency level)

5. рІерїЇрѕЇ рѕўрѕерїЃ рѕЏрѕхрїѕрЅБрЅх рІѕрІГрѕЮ рѕЏрІГрѕхрїѕрЅБрЅх рІГрѕЮрѕерїА / Choose:
   - рЅарѕхрѕЮ рѕўрѕхрїарЅх (With contact info)
   - рЅарѕхрѕЮ ріарІГрїѕрЅБ рѕўрѕхрїарЅх (Anonymous)

6. рІФрѕхрїѕрЅА / Submit

**рІерѕџріерЅ░рѕЅрЅхріЋ рѕЏрѕхрЅђрѕўрїЦ рІГрЅйрѕІрѕЅ:**
- рЇјрЅХ (Photos)
- рЅфрІхрІ« (Videos)
- рІхрѕЮрїй рЅЁрїѓ (Audio)

#### 2. рІеріцрѕхріцрѕЮріцрѕх / SMS Text Message

**рѕўрѕЇрІЋріГрЅх рѕўрї╗рЇЇ:**

```
TIP [рЅдрЅ│] | [рѕўрїЇрѕѕрїФ]
```

**рѕЮрѕ│рѕї / Example:**
```
TIP рЅдрѕї ріаріФрЅБрЅб | ріерЅБріЋріГ ріарїарїѕрЅЦ ріарїарѕФрїБрѕф ріЦріЋрЅЁрѕхрЅЃрѕ┤
TIP Bole area | Suspicious activity near bank
```

**рѕѕріарѕхрЅИрі│рІГ рѕЂріћрЅ│рІјрЅй / For urgent situations:**
```
TIP URGENT [рЅдрЅ│] | [рѕўрїЇрѕѕрїФ]
```

**рІерїЦрЅєрѕЏ рѕўрѕхрѕўрѕГ рѕхрѕЇріГ рЅЂрїЦрѕГ:** [рЅаріаріФрЅБрЅбрІј рІФрѕѕрІЇріЋ рЅЂрїЦрѕГ рІФрѕхрїѕрЅА]

#### 3. рІерѕхрѕЇріГ рїЦрѕф / Phone Call

**рІерЇќрѕірѕх рІерІхріЋрїѕрЅ░ріЏ рѕўрѕхрѕўрѕГ:**
- ріарѕхрЅИрі│рІГ: 911
- рІерїЦрЅєрѕЏ рѕўрѕхрѕўрѕГ: [рІеріаріФрЅБрЅб рЅЂрїЦрѕГ]

**рЅарѕхрѕЇріГ рїЦрѕф рѕѕрѕўрѕхрїарЅх:**
1. рІѕрІ░ рІерїЦрЅєрѕЏ рѕўрѕхрѕўрѕГ рІГрІ░рІЇрѕЅ
2. рѕѕрідрЇЋрѕгрЅ░рѕГ рІерІѕріЋрїђрѕЅріЋ рІЮрѕГрІЮрѕГ рІФрЅЦрѕФрѕЕ
3. рЅарѕхрѕЮ ріарІГрїѕрЅБ рѕЏрІхрѕерїЇ ріерЇѕрѕѕрїЅ рІФрѕ│рІЇрЅЂ

#### 4. рІерІхрѕе-рїѕрїй рЅЁрїй / Web Form

**рЅарібріЋрЅ░рѕГріћрЅх рЅаріЕрѕЇ:**
1. [www.crimeprevention.gov.et](http://www.crimeprevention.gov.et) рІГрїјрЅЦріЎ
2. "рїЦрЅєрѕЏ рІФрѕхрїѕрЅА" рѕІрІГ рІГрїФріЉ
3. рЅЁрї╣ріЋ рІГрѕЎрѕЅ
4. рІФрѕхрїѕрЅА

### рІерїЦрЅєрѕЏ рѕЏрѕхрїѕрЅбрІФ рѕўрѕўрѕфрІФрІјрЅй / Tip Submission Guidelines

#### рїЦрѕЕ рїЦрЅєрѕЏ рІерѕџріерЅ░рѕЅрЅхріЋ рІГрѕўрѕхрѕІрѕЇ:

РюЁ **рІЮрѕГрІЮрѕГ рѕўрѕерїЃ рІГрѕхрїА / Be Detailed:**
- рѕўрЅ╝ рЅ░ріерѕ░рЅ░? (When did it happen?)
- рІерЅх рЅ░ріерѕ░рЅ░? (Where did it happen?)
- рѕЏріЋ рЅ░рѕ│рЅ░рЇѕ? (Who was involved?)
- рѕЮріЋ ріарІГрЅ░рІЇ? (What did you see?)

РюЁ **ріЦрІЇріљрЅ░ріЏ рѕўрѕерїЃ рІГрѕхрїА / Be Accurate:**
- рЅЦрЅ╗ ріЦрѕГрїЇрїаріЏ рІерѕєріЉрЅхріЋ рІФрїІрѕЕ
- рІерЅ░рЇѕрїарѕерІЇріЋ рЅарЅхріГріГрѕЇ рІГрїЇрѕѕрї╣

РюЁ **рЅарїірІюрІЇ рІФрѕхрїѕрЅА / Be Timely:**
- рІѕрІ▓рІФрІЇріЉ рІФрѕхрЅ│рІЇрЅЂ
- рЇѕрїБріЋ рѕЮрѕІрѕй рІФрїѕріЏрѕЅ

РЮї **рІерѕЏрІГріерЅ░рѕЅрЅх / Avoid:**
- рІерІЇрѕИрЅх рѕўрѕерїЃ рѕўрѕхрїарЅх
- рѕ░рІјрЅйріЋ рЅарѕхрѕЮ рѕўріГрѕ░рѕх
- ріарѕІрѕхрЇѕрѕІрїі рѕўрѕерїЃ

### рІерІхрѕЁріљріљрЅх рїЦрЅарЅЃ / Privacy Protection

­Ъћњ **рЅарѕхрѕЮ ріарІГрїѕрЅБ рѕЏрІхрѕерїЇ / Anonymous Submissions:**
- рІеріЦрѕГрѕхрІј рѕўрѕерїЃ рЅарїГрѕФрѕй ріарІГрЅђрѕўрїЦрѕЮ
- рѕЏріЋріљрЅхрІј рѕЎрѕЅ рЅарѕЎрѕЅ рІГрїарЅарЅЃрѕЇ

­Ъћљ **рІерїЇрѕЇ рѕўрѕерїЃ / Personal Information:**
- рЅарѕхрѕЮ ріФрѕхрїѕрЅА рѕўрѕерїЃрІј рѕѕрЇќрѕірѕх рЅЦрЅ╗ рІГрѕєріЊрѕЇ
- рЅарѕџрѕхрїЦрѕГ рІГрїарЅарЅЃрѕЇ

### рІерїЦрЅєрѕЏ рІ░рѕерїЃрІјрЅй / Urgency Levels

| рІ░рѕерїЃ / Level | рѕўрїЇрѕѕрїФ / Description | рѕЮрѕ│рѕї / Example |
|-------------|-------------------|---------------|
| **Critical** | ріарѕхрЅИрі│рІГ рѕЮрѕІрѕй рІерѕџрІФрѕхрЇѕрѕЇрїЇ | рЅарѕѓрІ░рЅх рѕІрІГ рІФрѕѕ рІѕріЋрїђрѕЇ |
| **High** | рІерЇЇрїЦріљрЅх рѕЮрѕІрѕй рІерѕџрІФрѕхрЇѕрѕЇрїЇ | ріарїарѕФрїБрѕф рѕЂріћрЅ│ |
| **Medium** | рѕўрІ░рЅаріЏ рѕЮрѕІрѕй | рІерЅ░рЇѕрїарѕе рІѕріЋрїђрѕЇ рѕфрЇќрѕГрЅх |
| **Low** | ріарїарЅЃрѕІрІГ рѕўрѕерїЃ | рІерІѕріЋрїђрѕЇ ріарІЮрѕЏрѕџрІФ |

### рѕЮріЋ рІГріерѕ░рЅ│рѕЇ? / What Happens Next?

1. **рѕЏрѕерїІрїѕрїФ / Confirmation:**
   - рІѕрІ▓рІФрІЇріЉ рІерѕЏрѕерїІрїѕрїФ рѕўрѕЇрІЋріГрЅх рІГрІ░рѕГрѕхрІјрЅ│рѕЇ
   - рІерѕўрѕѕрІФ рЅЂрїЦрѕГ рІГрѕ░рїЦрІјрЅ│рѕЇ

2. **рїЇрѕЮрїѕрѕЏ / Review:**
   - рЇќрѕірѕх рїЦрЅєрѕЏрІјріЋ рІГрїѕрѕўрїЇрѕЏрѕЇ
   - рЅ░рїерѕЏрѕф рѕўрѕерїЃ рѕірїарІГрЅЂ рІГрЅйрѕІрѕЅ

3. **рѕЮрѕГрѕўрѕФ / Investigation:**
   - ріарѕхрЇѕрѕІрїі ріЦрѕГрѕЮрїЃрІјрЅй рІГрІѕрѕ░рІ│рѕЅ
   - рѕЂріћрЅ│рІЇ рІГріерЅ│рЅ░рѕІрѕЇ

4. **рІЇрїцрЅх / Resolution:**
   - рЅарѕхрѕЮ ріФрѕЇрѕєріљ рѕхрѕѕрІЇрїцрЅ▒ рІГрІ░рІЇрѕЅрѕЇрІјрЅ│рѕЇ
   - рѕЮрѕхрїІріЊ рІГрїѕрѕѕрїйрѕЇрІјрЅ│рѕЇ

## рѕѕрЇќрѕірѕх рѕ░рѕФрЅ░ріърЅй / For Police Officers

### рІерїЦрЅєрѕЏ ріарѕхрЅ░рІ│рІ░рѕГ / Managing Tips

#### рІерѕ░рѕїрІ│ рїѕрїй / Dashboard Access

1. рЅарѕхрѕГрІЊрЅ▒ рІГрїЇрЅА / Login to system
2. рІерѕЂрѕЅріЋрѕЮ рїЦрЅєрѕЏрІјрЅй рІГрѕўрѕЇріерЅ▒ / View all tips
3. рЅарѕџріерЅ░рѕЅрЅх рІФрїБрѕЕ / Filter by:
   - рѕЂріћрЅ│ (Status)
   - рІеріарІ░рїІ рІ░рѕерїЃ (Urgency)
   - рІерІѕріЋрїђрѕЇ ріарІГріљрЅх (Crime type)
   - рѕўрЅ░рѕІрѕѕрЇірІФ (Channel)

#### рІерїЦрЅєрѕЏ рѕЂріћрЅ│рІјрЅй / Tip Statuses

- **Pending** - ріарІ▓рѕх рїЦрЅєрѕЏ
- **Reviewing** - рЅарїЇрѕЮрїѕрѕЏ рѕІрІГ
- **Investigating** - рЅарѕЮрѕГрѕўрѕФ рѕІрІГ
- **Resolved** - рЅ░рЇѕрЅхрЅирѕЇ
- **Closed** - рЅ░рІўрїЇрЅирѕЇ

#### рІерѕџріеріЊрІѕріЉ рІхрѕГрїірЅХрЅй / Actions

1. **рѕЏрѕхрїѕрЅБрЅх / Assignment:**
   - рѕѕрідрЇірѕ░рѕГ рІГрѕўрІхрЅА
   - рѕѕрЅАрІхріЋ рІГрѕўрІхрЅА

2. **рѕЏрѕ╗рѕ╗рѕЇ / Update:**
   - рѕЂріћрЅ│ рІГрѕѕрІЇрїА
   - ріарѕхрЅ░рІФрІерЅх рІГрїерѕЮрѕЕ
   - рїЇрѕхрїІрѕ┤ рІФрЅЦрѕФрѕЕ

3. **рѕўрїѕріЊріўрЅх / Communication:**
   - рЅарѕхрѕЮ ріФрѕЇрѕєріљ рѕфрЇќрѕГрЅ░рѕЕріЋ рІФріЊрїЇрѕЕ
   - рЅ░рїерѕЏрѕф рѕўрѕерїЃ рІГрїарІГрЅЂ
   - рѕЮрѕхрїІріЊ рІФрѕхрЅ░рѕІрѕЇрЇЅ

### рІерѕфрЇќрѕГрЅх ріарїарЅЃрЅђрѕЮ / Reporting

## ріарїарЅЃрѕІрІГ рїЦрІФрЅёрІјрЅй / FAQs

**рїЦ: рЅарѕхрѕЮ ріарІГрїѕрЅБ рѕЏрІхрѕерїЇ рІ░рѕЁріЋріљрЅ▒ рІерЅ░рїарЅарЅђ ріљрІЇ?**
рѕў: ріарІјріЋрЇб рѕЏріЋріљрЅхрІј рЅарїГрѕФрѕй ріарІГрЅђрѕўрїЦрѕЮрЇб

**Q: Is anonymous submission safe?**
A: Yes. Your identity is never stored.

---

**рїЦ: рѕЮріЋ рІФрѕЁрѕЇ рїірІю рІГрІѕрѕхрІ│рѕЇ?**
рѕў: ріарѕхрЅИрі│рІГ рїЦрЅєрѕЏрІјрЅй рЅарІ░рЅѓрЅЃрІјрЅй рІЇрѕхрїЦ рІГрЅ│рІФрѕЅрЇб рѕўрІ░рЅаріЏрІјрЅ╣ рЅа24 рѕ░рІЊрЅх рІЇрѕхрїЦрЇб

**Q: How long does it take?**
A: Urgent tips are reviewed within minutes. Regular tips within 24 hours.

---

**рїЦ: рѕхрѕЁрЅ░рЅх ріФрІ░рѕеріЕ рѕЮріЋ рѕЏрІхрѕерїЇ ріарѕѕрЅЦріЮ?**
рѕў: ріЦріЋрІ░рїѕріЊ рїЦрЅєрѕЏ рІФрѕхрїѕрЅА рІѕрІГрѕЮ рІерїЦрЅєрѕЏ рѕўрѕхрѕўрѕГ рІГрІ░рІЇрѕЅрЇб

**Q: What if I made a mistake?**
A: Submit another tip or call the tip line.

---

**рїЦ: рЇјрЅХ рѕўрѕІріГ ріЦрЅйрѕІрѕѕрѕЂ?**
рѕў: ріарІјріЋрЇб рЅарѕърЅБрІГрѕЇ рѕўрЅ░рїЇрЅарѕфрІФ рІѕрІГрѕЮ рЅарІхрѕе-рїѕрїй рЅаріЕрѕЇрЇб

**Q: Can I send photos?**
A: Yes, through mobile app or web form.
